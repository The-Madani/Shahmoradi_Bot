# 🤖 ربات شاهمرادی

یه ربات تلگرام برای گروه‌ها با سیستم امتیاز، شرط‌بندی، بازی و هوش مصنوعی.
ربات شخصیت **استاد شاهمرادی** رو داره — یه معلم ریاضی افسانه‌ای که همیشه آماده جواب دادنه! 😄

> این فایل با کمک هوش مصنوعی نوشته شده.

---

## 📁 ساختار فایل‌ها

```
ShahmoradiBot/
│
├── main.py          # فایل اصلی — همه هندلرها اینجا ثبت میشن
├── config.py        # تنظیمات کلی (ادمین‌ها، کلید API، پارامترها)
├── database.py      # ذخیره و خواندن داده‌ها از JSON
├── commands.py      # دستورات عمومی و ادمین
├── betting.py       # سیستم شرط‌بندی
├── box_game.py      # بازی جعبه شانس
├── rps_game.py      # سنگ کاغذ قیچی
├── ai.py            # هوش مصنوعی Gemini (شخصیت استاد شاهمرادی)
├── events.py        # مدیریت درخواست عضویت
│
├── user_points.json   # ساخته میشه — امتیازات کاربران
├── user_levels.json   # ساخته میشه — لول کاربران
└── active_bets.json   # ساخته میشه — شرط‌های فعال
```

---

## 🚀 نصب و اجرا

```bash
pip install pyrogram tgcrypto google-genai
python main.py
```

قبل از اجرا باید تو `config.py` اینا رو تنظیم کنی:
- `GEMINI_API_KEY` — کلید API گوگل Gemini
- `ADMINS` — لیست آیدی عددی ادمین‌ها
- `MAIN_ADMIN_ID` — آیدی مدیر اصلی

## 🎮 دستورات

### برای همه:
| دستور | توضیح |
|-------|-------|
| `/start` | شروع |
| `/help` | راهنما |
| `/points` | امتیاز و لول خودت |
| `/leaderboard` | جدول برترین‌ها |
| `/bet [مقدار]` | شرط‌بندی روی تاس |
| `/cancelbet` | لغو شرط فعال |
| `/mybets` | شرط فعالت |
| `/rps` | سنگ کاغذ قیچی |
| `/ai [سوال]` | سوال از استاد شاهمرادی |
| `/movie [نام]` | جستجو فیلم و سریال |

### برای ادمین‌ها:
| دستور | توضیح |
|-------|-------|
| `/addpoints [مقدار]` | اضافه کردن امتیاز (روی پیام ریپلای کن) |
| `/removepoints [مقدار]` | کم کردن امتیاز |
| `/setpoints [مقدار]` | تنظیم مستقیم امتیاز |
| `/pending` | درخواست‌های عضویت در انتظار (فقط مدیر اصلی) |

## ✨ ویژگی‌ها

### 💎 سیستم امتیاز و لول
- **8 لول** از نوب تا خدا
- امتیاز از تاس، جعبه شانس، سنگ کاغذ قیچی
- نوار پیشرفت و اعلان لول‌اپ

| لول | عنوان | امتیاز |
|-----|-------|--------|
| 1 | 🔰 نوب | 0 - 99 |
| 2 | 🔗 نوپا | 100 - 249 |
| 3 | ⭐ ستاره | 250 - 499 |
| 4 | 🔥 حرفه‌ای | 500 - 999 |
| 5 | 💎 الماسی | 1000 - 1999 |
| 6 | 👑 پادشاه | 2000 - 4999 |
| 7 | 🌟 افسانه‌ای | 5000 - 9999 |
| 8 | 🏆 خدا | 10000+ |

### 🎲 شرط‌بندی
تاس بزن، شرط ببند، امتیاز ببر یا بباز:

| نوع شرط | ضریب |
|---------|-------|
| زوج / فرد | ×2 |
| محدوده 1-3 یا 4-6 | ×3 |
| عدد دقیق | ×6 |

### 📦 جعبه شانس
هر 50 پیام تو گروه یه سوال ریاضی میاد. جواب درست بدی 5 تا 15 امتیاز میگیری.

### ✊ سنگ کاغذ قیچی
با `/rps` بازی کن — برد +15، باخت -5، مساوی بدون تغییر.

### 🤖 هوش مصنوعی
با `/ai [سوال]` از استاد شاهمرادی بپرس. پاسخ‌ها با Gemini 2.5 Flash تولید میشن.

## 🐛 مشکلات رایج

**ربات استارت نمیشه:**
فایل session رو چک کن. اگه نداری باید یه بار با شماره لاگین کنی تا ساخته بشه.

**AI جواب نمیده:**
کلید `GEMINI_API_KEY` رو چک کن. از [Google AI Studio](https://aistudio.google.com) میتونی بگیری.

**جعبه ظاهر نمیشه:**
باید دقیقاً 50 پیام متنی (نه دستور) تو گروه فرستاده بشه.
